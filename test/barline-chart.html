Line Shadow Options

  • shadowColor color of line shadows black.

  • shadowBlur the amount of blurring effect of line shadows. The default value is 4.

  • shadowOffsetX horizontal distance the line shadows will be offset. The default value is 2.

  • shadowOffsetY vertical distance the line shadows will be offset. The default value is 2.
